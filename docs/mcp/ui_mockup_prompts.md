# MCP UI 목업 생성 프롬프트 (5개 화면)

> **목적**: MCP 이미지 생성 도구를 활용한 체크 알리미 앱 UI 목업 프롬프트
> **디자인 컨셉**: 미니멀, 한국어 UI, iOS/Android 공통, Material Design 3 기반

---

## 1. Home 화면 (오늘의 체크 리스트)

### 프롬프트

```
한국어 모바일 앱 UI 목업을 생성해주세요.

화면: "오늘의 체크" 홈 화면 (체크 알리미 앱)
디자인 스타일: Material Design 3, 미니멀, 밝은 테마

상단 영역:
- 앱바: "체크 알리미" 타이틀 (좌측), 프로필 아이콘 (우측)
- 오늘 날짜 표시: "2026년 2월 7일 (토)" 큰 텍스트
- 원형 진행률 표시: 중앙에 큰 원형 프로그레스 (예: 3/5 완료, 60%)
- 진행률 아래 텍스트: "5개 중 3개 완료"

체크 리스트 영역:
- 세로 스크롤 리스트, 카드 형태
- 각 항목:
  - 좌측: 아이콘 (💧, 🏃, 📚, 💊, 🧘)
  - 중앙: 항목 이름 ("물 마시기", "운동 30분", "독서 20분", "비타민 섭취", "명상 10분")
  - 우측: 원형 체크박스 (완료 시 색상 채움 + 체크마크)
  - 완료된 항목: 이름에 취소선, 배경 살짝 어둡게
- 3개 완료, 2개 미완료 상태

하단 네비게이션 바:
- 4개 탭: "오늘" (활성), "항목 관리", "통계", "설정"
- 각 탭에 아이콘 + 라벨

색상: 메인 컬러 #4CAF50 (초록), 배경 #FAFAFA, 카드 #FFFFFF
폰트: 한국어 (Pretendard 스타일)
디바이스: iPhone 15 Pro 프레임
```

## 2. Task List 화면 (항목 관리 목록)

### 프롬프트

```
한국어 모바일 앱 UI 목업을 생성해주세요.

화면: "항목 관리" 리스트 화면 (체크 알리미 앱)
디자인 스타일: Material Design 3, 미니멀, 밝은 테마

상단 영역:
- 앱바: "항목 관리" 타이틀 (좌측), 검색 아이콘 + "+" 추가 버튼 (우측)

카테고리 필터 칩:
- 가로 스크롤: "전체" (활성), "건강", "운동", "학습", "생활"

항목 리스트:
- 드래그 가능 리스트 (좌측에 핸들 아이콘 ≡)
- 각 항목 카드:
  - 좌측: 색상 원 + 아이콘 (예: 초록 원 안에 💧)
  - 중앙:
    - 항목 이름 (굵게): "물 마시기"
    - 부제: "매일 · 오전 9:00 알림"
  - 우측: 토글 스위치 (활성/비활성)
- 5-6개 항목 표시
- 비활성 항목은 회색 처리

플로팅 액션 버튼:
- 우하단 FAB: "+" 아이콘 (메인 컬러)

하단 네비게이션 바:
- 4개 탭: "오늘", "항목 관리" (활성), "통계", "설정"

색상: 메인 컬러 #4CAF50, 카테고리 칩별 다른 색상
디바이스: iPhone 15 Pro 프레임
```

## 3. Task Detail 화면 (항목 추가/수정 폼)

### 프롬프트

```
한국어 모바일 앱 UI 목업을 생성해주세요.

화면: "항목 추가" 폼 화면 (체크 알리미 앱)
디자인 스타일: Material Design 3, 미니멀, 밝은 테마

상단 영역:
- 앱바: "←" 뒤로가기 (좌측), "항목 추가" 타이틀 (중앙), "저장" 텍스트 버튼 (우측, 메인 컬러)

폼 내용 (세로 스크롤):
1. 항목 이름 입력:
   - 라벨: "항목 이름"
   - Material 텍스트 필드, 플레이스홀더: "예: 물 마시기"

2. 아이콘 선택:
   - 라벨: "아이콘"
   - 가로 스크롤 이모지 그리드 (4x3): 💧🏃📚💊🧘🎵🍎✏️🌱🎯💪📝
   - 선택된 아이콘에 테두리 표시

3. 색상 선택:
   - 라벨: "색상"
   - 가로 색상 원 8개 (빨강, 주황, 노랑, 초록, 파랑, 남색, 보라, 핑크)
   - 선택된 색상에 체크마크

4. 카테고리 선택:
   - 라벨: "카테고리"
   - 드롭다운 또는 칩 그룹: "건강", "운동", "학습", "생활", "기타"

5. 반복 요일 선택:
   - 라벨: "반복 요일"
   - 7개 원형 버튼 가로 배치: "월 화 수 목 금 토 일"
   - 선택된 요일은 메인 컬러로 채움
   - "매일" 토글 버튼

6. 알림 시간:
   - 라벨: "알림 시간"
   - 시간 선택 UI: "오전 9:00" (탭하면 타임피커)

색상: 메인 컬러 #4CAF50
디바이스: iPhone 15 Pro 프레임
```

## 4. Statistics 화면 (통계/캘린더 히트맵)

### 프롬프트

```
한국어 모바일 앱 UI 목업을 생성해주세요.

화면: "통계" 화면 (체크 알리미 앱)
디자인 스타일: Material Design 3, 미니멀, 밝은 테마

상단 영역:
- 앱바: "통계" 타이틀 (좌측)
- 월 선택: "◀ 2026년 2월 ▶" (좌우 화살표로 월 변경)

요약 카드 (가로 스크롤 또는 3열 그리드):
- 카드 1: "완료율" - "78%" (큰 텍스트, 초록)
- 카드 2: "연속 달성" - "12일" (큰 텍스트, 파랑)
- 카드 3: "총 완료" - "156회" (큰 텍스트, 주황)

캘린더 히트맵 영역:
- 라벨: "월별 달성도"
- 월간 캘린더 그리드 (7열 x 5행)
- 요일 헤더: "월 화 수 목 금 토 일"
- 각 날짜 셀의 배경색으로 달성률 표시:
  - 0%: 회색 (#E0E0E0)
  - 1-33%: 연한 초록 (#C8E6C9)
  - 34-66%: 중간 초록 (#81C784)
  - 67-99%: 진한 초록 (#4CAF50)
  - 100%: 매우 진한 초록 (#2E7D32) + 작은 체크마크
- 오늘 날짜에 테두리 강조
- 하단 범례: 회색→진한초록 그라데이션 바 + "0% ~ 100%"

항목별 통계 리스트:
- 각 항목 카드:
  - 아이콘 + 이름: "💧 물 마시기"
  - 미니 프로그레스 바: 85% (초록)
  - 텍스트: "이번 달 26/28일 완료"

하단 네비게이션 바:
- 4개 탭: "오늘", "항목 관리", "통계" (활성), "설정"

색상: 히트맵 초록 계열, 메인 컬러 #4CAF50
디바이스: iPhone 15 Pro 프레임
```

## 5. Daily Check 화면 (날짜별 체크 상세)

### 프롬프트

```
한국어 모바일 앱 UI 목업을 생성해주세요.

화면: "날짜별 체크" 상세 화면 (체크 알리미 앱)
디자인 스타일: Material Design 3, 미니멀, 밝은 테마

상단 영역:
- 앱바: "←" 뒤로가기 (좌측), "날짜별 기록" 타이틀 (중앙)
- 주간 캘린더 스트립:
  - 가로 스크롤, 7일 표시
  - 각 날짜: 요일 약자 + 날짜 숫자 (예: "월 3", "화 4", ...)
  - 선택된 날짜: 메인 컬러 원형 배경
  - 오늘: 테두리 강조
  - 완료율이 높은 날: 아래 작은 초록 점

날짜 정보:
- 큰 텍스트: "2월 7일 토요일"
- 서브텍스트: "4/5 완료 (80%)"
- 가로 프로그레스 바: 80% 채움

체크 리스트 (해당 날짜):
- 세로 리스트, 타임라인 스타일
- 각 항목:
  - 좌측: 세로 타임라인 라인 + 원형 체크 아이콘
  - 중앙:
    - 항목 이름: "💧 물 마시기"
    - 완료 시각: "오전 9:15에 완료" (초록 텍스트)
    - 또는 "미완료" (회색 텍스트)
  - 우측: 체크/미체크 상태 아이콘
- 완료된 항목: 체크마크 + 완료 시각
- 미완료 항목: 빈 원 + "미완료" 텍스트
- 시간순 정렬

메모/한줄평 섹션 (선택적):
- 카드: "오늘의 한마디"
- 텍스트 입력 필드: "오늘 4개나 완료! 잘했다 💪"

색상: 메인 컬러 #4CAF50, 타임라인 라인 #E0E0E0
디바이스: iPhone 15 Pro 프레임
```

| Primary Light | 연한 초록 | #C8E6C9 |
| Secondary | 파랑 | #2196F3 |
| Accent | 주황 | #FF9800 |
| Background | 밝은 회색 | #FAFAFA |

| Error | 빨강 | #F44336 |

| 캡션/시간 | 12sp | Regular |
